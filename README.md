**Birthday-Wisher**

**Introduction:**

The Birthday Wishes Emailer is a Python script designed to automate the process of sending birthday wishes via email. Leveraging the `smtplib`, `pandas`, and `ssl` modules, the script retrieves birthday data from a CSV file, checks for matches with today's date, and sends personalized emails to the respective individuals. 

**Features:**

- Automates the process of sending birthday wishes via email.
- Reads birthday data from a CSV file.
- Sends secure emails using SMTP and SSL.
- Allows customization of email content with random text files.
- Personalizes emails with the recipient's name.

**Installation:**

1. Download the project repository from PythonAnywhere.
2. Install the required dependencies: `smtplib`, `pandas`, and `ssl`.
3. Ensure Python is installed on your system.

**Usage:**

1. Modify the sender's email and password to your own credentials.
2. Generate an application-specific password for Gmail and update the script accordingly.
3. Verify file paths and configurations.
4. Run the script, ensuring it is scheduled to run at the desired time (e.g., using PythonAnywhere).
